This project requires Python 2.7 and the following Python libraries installed:

### Technology Stack

### Data Exploration:

- Numpy
- Pandas

### Data visualization:

- Matplotlib
- Altair
- Seaborn

### Data Preparation:

- nltk

### Modeling:  


You will also need to have software installed to run and execute an iPython Notebook
